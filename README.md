# Factorización en Python

Este proyecto implementa:

- Algoritmo de Fermat
- Factorización por reducción de raíz cuadrada

## Descripción

Fermat es eficiente cuando los factores son cercanos.
La reducción por raíz cuadrada es útil cuando los factores están alejados.

## Estructura

- `fermat.py`: algoritmo de Fermat
- `sqrt_reduction.py`: factorización clásica
- `experimentos.py`: pruebas comparativas

## Ejecución

```bash
python experimentos.py
